# encryptCamera
An encrypted camera android software
Brief introduction
At present, many users hope to be kept privately when they take photos,but the Information leakage of mobile phones are very serious. Some mobile phone applications will steal usersâ€™ data, which causes great troubles for the majority of users. Our product can directly solve this problem. For those in the blockchain industry, encrypted camera can also help users save private keys without cumbersome process of paper backup.
